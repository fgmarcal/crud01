# Projeto CRUD com Node.js

## API de Agenda de Contatos

### Usuário
- Criar usuário
- Proteger duplicação de usuários por meio do e-mail

### Contatos
- criar contatos
- alterar contatos
- listar contatos
- apagar contatos

### Tecnologias utilizadas

- Node
- Typescript
- Prisma
- Fastify
- Sqlite

## Arquivo .env
- crie um arquivo .env na pasta mãe com os dados abaixo:
    - DATABASE_URL="file:./dev.db"
